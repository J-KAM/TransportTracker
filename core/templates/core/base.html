Help

Search text is not predefined. User can write whatever he wants but query elements must be specified in the following order:

  1. Number of tickets
  2. Ticket type
  3. Ticket class (only for plane and train tickets)
  4. Transport type
  5. Departure and arrival cities
  6. Departure date
  7. Return date (only for round-trip tickets)
  8. Max price per person and currency (optional)

Number of tickets

Must be a positive number. It cannot be written in letters, only digits are accepted as valid number.

Ticket type

User must define whether he needs a one-way ticket or a round-trip ticket.

Ticket class

Depends on the mean of transport. For plane it may be economy or business class. Train classes are: first, second, couchette.

Transport type

Means of transport are plane, bus or train.

Departure and arrival cities

Order of these cities is not imposed. Keyword for departure city is from and for arrival city is to. Valid city names contain only letters. If a city name contains more than one word you should separate them with - instead of whitespace.

Departure date

Key word for this date is on. Accepted date formats are: dd/mm/yyyy and dd.mm.yyyy

Return date

Key word for this date is return. Accepted date formats are: dd/mm/yyyy and dd.mm.yyyy

Max price and currency

If you wish to limit maximum ticket price you should specify your upper price limit. Keyword you must use is price. It cannot be written in letters, only digits are accepted as valid price. You should also specify price currency. Avaliable currencies are EUR, RSD and USD.

Examples

Valid plane queries

  • I need 2 one-way economy class plane tickets from Manchester to Amsterdam on 19/02/2018. Maximum ticket price is 400 eur.
  • I would like to buy 2 round-trip tickets. Desired class is economy. I want to travel by plane from Belgrade to Milan on 15/03/2018 with the return date 20/03/2018.

Valid train queries

  • We want 5 one-way first class train tickets from Vienna to Prague on 03/06/2018. Maximum ticket price is 20 usd.
  • We want 3 round-trip second class train tickets from Paris to Lyon on 01/04/2018. I want to return on the 06/04/2018.

Valid bus queries

  • We want 1 one-way bus ticket from Vienna to Novi-Sad on 03/06/2018. Maximum ticket price is 10000 rsd.
